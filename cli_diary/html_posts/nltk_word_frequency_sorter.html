Sort Text by Word Frequency With NLTK

30/04/2022

I found NLTK, ‘Natural Language Toolkit’, when I was looking for something to extract keywords from text. Off course at the time I didn’t knew that keyword extraction is called keyword extraction, so I was looking for tag creation which, I found out, is a totally different story, but led me to an article that suggested word frequency analysis as a method to generate keywords. I now know that this is totally inadequate for this purpose but it can have merit on other uses; and for that I’m documenting it here.
I’ll use as an example, a module I wrote to bulk import the contents of a blog to a database. I wanted something that would automate the creation of tags for the content, so I tried NLTK.

For each post file I would read its contents, strip it of line break symbols and turn it into a string, as that is the accepted format for NLTK.

        with open(file, "r") as f:
            da = f.readlines()
            dat = [i.strip() for i in da]
            data = str(dat)

I defined a list of characters to ignore on the text analysis…

       badchars = ".!?,'\":<>="

created a list of words longer than four characters, striped of said characters:

       words = [word.strip(badchars) for word in data.strip().split() if len(word) > 4]

create a dictionary with the words and a numerical key:

       word_freq = {}
       for word in words:
           word_freq[word] = word_freq.get(word, 0) + 1

create a list of tuples made up of the words and their id:

      tx = [(v, k) for (k, v) in word_freq.items()]

sort list in descending order:

      tx.sort(reverse=True)

and present the tuple list with the values first.

      word_freq_sorted = [(k, v) for (v, k) in tx]

The end result is a list with a lot, and I mean a lot, of trash in the form of useless words, punctuations marks and other assorted artefacts that contribute nothing to the task at hand.
For this particular case, this is the ‘cleaning’ commands I had to give, and it didn’t get as clean as I would like.
To erase all entries in a list containing a specific character, first create a list with all entries that have the character, then create a new list from the original list that excludes the entries on the latter.
